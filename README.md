# YourBase CLI

A build tool that makes working on projects much more delightful.

# How to use

1. Download 
2. Clone a package
3. Build

# Example commands

(Alias artificer as `yb` or copy it to a file with that name in your PATH) 

* `git clone https://github.com/johnewart/simple-java-app.git`
* `cd simple-java-app`
* `yb build`
